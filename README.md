# Description
Personal website! Moved from old repository to new, fixed code readability and implemented new features (mobile compatibility, interactive theme toggle)

## Tech Stack
- HTML used to build pages of the website
- CSS used for custom layout and UI
- Javascript used for interactive elements (theme button toggle, hamburger menu)

## Motivation
After discovering my website completely breaks on mobile, I decided to fix this issue by actively testing different ways to make my website mobile-friendly, and the solution I found was to collapse the navbar into a hamburger menu on small screens
